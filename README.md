## 프로그램 실행법

### <주의> 본 DApp은 Ropsten 테스트 네트워크에 배포됐습니다.
* Node.js 웹 서버 실행
```
node server.js
```
* 클라이언트 접속
```
웹 브라우저 실행 > 127.0.0.1:8080/index.html 접속
```

* DApp 사용
```
1. HTML 로드 시 메타마스크 연결
2. UI를 통해 등록/조회 등 수행
```
